## 更新日志

### V1.0 版本

- 功能
  - 将键盘输入两个数进行对应的操作【加减乘除】

- 新功能

  - 以抽象和多态方式实现加、减、乘、除运算

### V1.2 版本

- 新功能
  - 对代码进行了重构，通过接口新增对用户输入的两个字符串进行加减操作
